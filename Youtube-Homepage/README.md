🎥 YouTube Homepage Clone (HTML & CSS)

This is a basic YouTube homepage clone built using HTML and CSS as part of my frontend learning journey.

🚀 About the Project

This project focuses on building a simple layout similar to YouTube’s homepage using core CSS concepts.
It is not a full replica, but a practice project to understand how real-world layouts are structured.

🛠️ What I Used
HTML
CSS
Flexbox
Grid
Media Queries
📚 What I Learned
How to structure layouts using Flexbox and Grid
Basics of responsive design using media queries
Managing spacing, alignment, and positioning
Planning a layout before coding (deciding where to use Flexbox vs Grid)
💡 Key Takeaways

One important thing I learned from this project is the value of planning before building.
Even for a simple project, deciding layout structure (Flexbox/Grid), spacing, and sections in advance makes the process much smoother.

⚠️ Note

Some parts of the styling are still incomplete (like responsiveness in certain sections), and I plan to improve them in the future.
